CAMPUS GIG

The Student Job Application System is a simple Java-based desktop
application that connects students with employers. It allows students to
create profiles, search for jobs, and enables employers to post job
opportunities. An admin panel provides an overview of registered users.

Features

 Student

-   Create a profile (name, email, skills)
-   Search for jobs based on skills and location

 Employer

-   Create an employer account
-   Post job opportunities

 Admin

-   View all registered students
-   View all employers and jobs posted

 Project Structure

studentjobapp/ - StudentJobApp.java - StudentMenu.java -
EmployerMenu.java - AdminMenu.java

Technologies Used

-   Java
-   Java Swing (JOptionPane)
-   ArrayList & List

 How to Run

1.  Open project in an IDE (NetBeans, IntelliJ, Eclipse)
2.  Ensure all files are in the same package: studentjobapp
3.  Run StudentJobApp.java

 Known Issues

-   No data persistence
-   No authentication
-   Jobs list may not be initialized

 Suggested Improvements

-   Add database storage
-   Implement login system
-   Improve UI
